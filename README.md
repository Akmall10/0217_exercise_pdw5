# ✦ Kopi Langit — Coffeeshop Yogyakarta

> *"Tempat di Mana Kopi Bertemu Jiwa"*

Website profil coffeeshop premium fiktif bernama **Kopi Langit** yang berlokasi di Jalan Prawirotaman, Yogyakarta. Dibuat dengan HTML, CSS murni, dan JavaScript vanilla tanpa framework.

---

## 🎯 Tema Project

**Coffeeshop Artisan di Yogyakarta** — Kopi Langit adalah kedai kopi yang menggabungkan estetika luxury editorial dengan kehangatan budaya Jogja. Website ini menampilkan brand story, menu signature, galeri suasana, dan informasi kontak.

**Arah Desain:** Dark luxury editorial — hitam pekat, aksen emas, tipografi serif dramatis (Cormorant Garamond), animasi sinematik.

## ✨ Fitur Desain

- **Custom cursor** emas yang mengikuti mouse
- **Navbar fixed** yang berubah saat scroll
- **Hero fullscreen** dengan animasi zoom & teks besar dramatis
- **Marquee strip** teks berjalan otomatis
- **Section Tentang** dua kolom dengan kartu statistik mengambang
- **Menu grid** 3 kolom dengan hover zoom + badge
- **Galeri masonry** dengan overlay label
- **Pull quote** sinematik dengan tanda kutip besar
- **Footer** elegan dengan CTA reservasi
- **Fully responsive** untuk mobile & tablet
